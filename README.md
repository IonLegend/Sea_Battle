# SeaBattle 

Чистая Python-библиотека для игры в морской бой.

## 🧩 Зачем эта библиотека

В открытых реализациях морского боя на Python часто одно из двух:
- либо **игру нельзя использовать как библиотеку** — логика жёстко привязана к консоли или GUI
- либо **API для AI отсутствует** — чтобы написать умного противника, приходится лезть во внутренности

SeaBattle решает эти проблемы:
- **чистое разделение логики и интерфейса** — используйте в консоли, GUI, телеграм-боте или как основу для AI-соревнований
- **готовый API для AI** — `get_available_cells()`, `get_neighbours()`, `get_remaining_ships_lengths()` — всё для вероятностной стрельбы и охоты
- **автоматическое управление состоянием** — выстрел сам обновляет доступные клетки и историю, AI не нуждается в дополнительном хранилище

Библиотека берёт на себя всю рутину: валидацию, буферные зоны, контроль флота. Вы сосредотачиваетесь на стратегии, а не на поддержании порядка на поле.


## 🔥 Почему это удобно

- **👌 Человеческие координаты**  
  Вместо `(0,0)` пишете `'a1'`. Удобно для отладки и для AI.

- **⚙ Автоматическая буферная зона**  
  Корабли не касаются даже углами. Библиотека следит за этим сама.

- **🤖 API для AI**  
  `get_available_cells()`, `get_neighbours()`, `get_remaining_ships_lengths()` — всё, что нужно для умного противника.

- **🔍 Состояние под контролем**  
  Выстрел сам обновляет доступные клетки, историю и состояние кораблей. AI не нужно ничего хранить.

- **📃 Один файл**  
  Скопировал — и работает. Без зависимостей и сложной установки.

- **📁 Документация и тесты**  
  У каждого метода есть docstring. 28 тестов проверяют ключевые функции.

## Быстрый старт

```python
from seabattle_core import Field

field = Field()
field.random_placing()
result = field.shot('a1')
print(result)

if field.is_game_over():
    print("Игра окончена!")
```

## 🤖 API для AI

| Метод | Описание |
|-------|----------|
| `get_available_cells()` | Список клеток для расстановки кораблей |
| `get_neighbours(coord)` | Соседние клетки |
| `cell_state(coord)` | Состояние клетки |
| `get_remaining_ships_lengths()` | Длины оставшихся кораблей |
| `coord_to_index(coord)` | `'a1'` → `(0,0)` |
| `index_to_coord(x, y)` | `(0,0)` → `'a1'` |
